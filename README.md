# Sales Data Analysis Dashboard

## Project Overview
This project analyzes sales data using Python and visualizes product performance and profit trends.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Total sales analysis
- Product-wise sales insights
- Profit analysis by category
- Data visualization using bar charts

## How to Run
1. Install libraries:
   pip install pandas matplotlib

2. Run:
   python main.py
